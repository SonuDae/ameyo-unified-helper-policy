# Privacy Policy for Ameyo Unified Helper

**Last Updated: May 2026**

Ameyo Unified Helper operates entirely as a local browser utility designed to optimize user workflows, streamline data verification, and enhance productivity. 

### Data Collection & Usage
* **No Personal Data Collection:** This extension does not collect, harvest, or transmit any personally identifiable information, browsing histories, or user activities to any remote servers.
* **Website Content Permissions:** The extension interacts with specific webpage interface elements locally on your device solely to execute features like text copying, status checking, and interface macros as requested by the user.
* **Local Storage:** Any configurations or preferences are saved locally on your machine using standard browser storage permissions and are never shared.

### Contact
For any policy inquiries, please reach out via the official developer contact channel on the Chrome Web Store listing.
